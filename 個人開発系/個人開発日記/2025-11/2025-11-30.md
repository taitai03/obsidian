## 同期処理と非同期処理の違い
### 同期処理(Synchronous)
1. 一つずつ順番に実行。コードは上から順にひとつずつに実行
2. もし途中の操作が時間かかるもんであった場合、その操作は<font color="#ff0000">待機</font>される
3. 上記の待機のため、時間画家から操作中はアプリケーション全体が「ブロック」され、他の操作を受け付けないことがある。

### 非同期処理(Asynchonous)
1. 同時に複数の処理。一つの操作の完了を待たずに次の操作が始まる事ができます。
2. コールバックやプロミス：非同期の操作が完了したら、コールバック関数を実行したり、Promissが解決または拒否されることでmその結果を取り扱う事ができる
3. 非ブロッキング：非同期処理はアプリケーションをブロックしないので、ネットワークリクエストを待っている間もユーザーインターフェースは反応し続ける事ができる。


## Expo SQLiteの使い方
### 基本的なCRUD操作
①データを取り出す準備
```typescript
import * as SQLite from 'expo-sqlite';

const db = await SQLite.openDatabaseAsync('databaseName');
```
②まとめてクエリを実行したいとき
```typescript
await db.execAsync(` PRAGMA journal_mode = WAL; CREATE TABLE IF NOT EXISTS test (id INTEGER PRIMARY KEY NOT NULL, value TEXT NOT NULL, intValue INTEGER); INSERT INTO test (value, intValue) VALUES ('test1', 123); INSERT INTO test (value, intValue) VALUES ('test2', 456); INSERT INTO test (value, intValue) VALUES ('test3', 789); `);
```
③ひとつずつ実行したいとき
```typescript
const result = await db.runAsync('INSERT INTO test (value, intValue) VALUES (?, ?)', 'aaa', 100);
```
