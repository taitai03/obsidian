# 数値計算の基本
条件分岐に関する注意：浮動小数点に関する条件式は数値誤差を十分に考慮する必要がある。特に、<span style="background:#fff88f">浮動小数点の比較で、「＝＝」を使って完全一致をし調べるのは悪手である</span>

誤差が比較的小さく抑えられる解法を、<font color="#ff0000">数値的に安定な解法</font>と呼ぶ

数値計算するときは、以下の点に注意する
- 原則として、浮動小数点型の比較には、`==`や`!=`などの<span style="background:#fff88f">完全一致するかの比較は行わない</span>ようにする
- 数値が近いものの引き算は、<span style="background:#fff88f">桁落ち現象</span>により、有効桁数が失われるので注意
- 計段結果が計算機内で扱えるはずなのに、途中で<span style="background:#fff88f">infや-infが生じてしまう計算</span>には注意


# Numpyの基本

eのべき乗:`np.exp(2)`

自然対数：`np.log(np.e)`

正弦：`np.sin(np.pi)`

平方根：`np.sqrt(3)`


# Numpyの配列


引数を三つ与える
