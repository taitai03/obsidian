## str関数int関数
str()は、入力された値を文字列型にする
int()は、入力された値を数値型にする

次のように "C:\test" をそのまま出力すると、`\t` がタブとして動作してしまい、期待通りに表示されません。
```python
print("C:\test")
```
回答例のようにエスケープシーケンス `\\` を使用することで、文字列内に `\` を文字として含めることができます。  
このように、エスケープシーケンスを通常の文字として扱うための操作を `エスケープ` と呼びます。  
一部の文字列に特殊な意味を持たせるエスケープシーケンスに対し、その特殊な動作を防ぐのがエスケープです。

**エスケープシーケンス**：\nなどの特殊な文字列のこと

r文字列を扱うことで、エスケープシーケンスを無効化し、記載した文字列をそのままの形で扱うことができる。しかし、文字列の末尾に\の記号があると、エラーになる場合があるから注意する。

## f文字列の使用
```python 
name="Python"
print(f"Hello,{name}!")
```
f文字列を使って、{}内に変数目を書くことで、文字列の中に変数の値を埋め込むことができる。

## 部分文字列
```python
x="Python"
print(x[0:2])
```
文字列の範囲指定は`[開始インデックス:終了インデックス]`の形式でするが、<span style="background:#fff88f">最終インデックスは出力されない</span>ことに注意
