# Punditについて

attr_reader:インスタンス変数の読み取り専用のメソッドを自分で定義することができるメソッド

前提：Rubyはクラスのインスタンス変数を外部から参照する場合には、読み取り専用メソッドを以下のように定義する必要がある
```Ruby
class User
  def initialize(name, age)
    @name = name
    @age = age
  end

  def name # 読み取り専用のメソッド
    @name
  end

  def age # 読み取り専用のメソッド
    @age
  end
end

tanaka = User.new('田中太郎', 18)
p tanaka.name # @nameを読み取る
#=> "田中太郎"
p tanaka.age # @ageを読み取る
#=> 18
```
しかし、これだと<span style="background:#fff88f">参照したいインスタンスが増えるたびに読み取り専用メソッドを定義する必要</span>がある。
そこで、attr_readerを使うと、以下のように記述するコードの量を減らすことができる
```Ruby
class User
  attr_reader :name, :age # これで読み取り専用のメソッドが定義される

  def initialize(name, age)
    @name = name
    @age = age
  end
end

tanaka = User.new('田中太郎', 18)
p tanaka.name # @nameを読み取る
#=> "田中太郎"
p tanaka.age # @ageを読み取る
#=> 18
```
