# SQL
### 特定の複数のカラムをテーブルから選択
``` SQL
SELECT business_category, daily_revenue FROM daily_retail_data
```
### 降順、昇順
降順：DESC(descenfding)
昇順：ASC(ascending)

## データの件数をカウント
```SQL
SELECT COUNT(*) as total_records
FROM daily_retail_data;
```
### 単純な条件の絞り込み
```SQL
WHERE daily_revenue >= 50000;
```
### 重複を除いた一覧表示
```SQL
SELECT DISTINCT business_category FROM daily_retail_data
```
