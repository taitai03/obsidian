### javascriptの基礎中の基礎

アロー関数
```javascript
//通常の書き方
function add(a,b){
	return a+b;
}

//アロー関数
const add = (a,b)=>a+b;
```
map
```javascript
const nums=[1,2,3]
const doubled=nums.map(n=>n**2) //[2,4,6]
```

fileter配列から「条件に合うものだけ」残す
```javascript
const nums=[1,2,3,4]
const even=nums.filter(n => n%2 ===0); //[2,4]
```
スプレット構文
```javascript
const arr=[1,2,3]
const copy=[...arr]

const user={name: "太郎"}
const newUser={...user, age;26}
```
分割代入
```javascript
const user={name: "太郎",age:26};
const {name,age}=user;
```
