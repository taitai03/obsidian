## 機械学習、ディープランニング、人工知能の違い
機械学習は、まず、学習させるためのモデルの選択、次に人の手によって特徴を選択
ディープランニングは、まず、ネットワークの方を決めて、演算が行われた結果、自動的に特徴が抽出される

### 機械学習の例
ゲームの体の動きの認知などでも使われている。音声認識は機械学習によって精度が高くなっている

## 独立変数と従属変数に分ける理由
- アルゴリズムが変数をどう扱えばいいかわからないから分ける

### ilocメソッド（アイロックメソッド）
数字の指定することによって対象のデータを絞り込む
```python
x=dataset.iloc[:,:-1].values
			#[行,列]を表す
y=dataset.iloc[:,-1].values
```
:の指定は全てのデータを意味する
:と数字の場合、その数字より前のデータを示す


### そもそも機械学習とは何をするか
 ：データをもとに、そのデータから何かしらの特徴を抽出する
- データがなければ何も始まらない
- データをインポートするときは都度新しい変数名に格納してあげるのが一般的
- そうしないと元のデータに戻すのが大変

### 欠損値の処理
- 適切な処理をしないと分析できない
- 一番簡単な操作は欠損している部分のデータを削除してしまうこと
- 一般的には全体の1％未満なら消しても良い
- ただ、それだけだと対処できないので、保管するという考え方がある（平均値、最頻値、中央値の三つの取り方がある）

# SQL関連
## LIKE:検索する時に用いる


# 1.前方一致
```SQL
SELECT user FROM users WHERE user LIKE "山%";
```
# 2.後方一致
```SQL
SELECT user FROM users WHERE user LIKE "%山";
```
# 3.部分一致検索
```SQL
SELECT user FROM users WHERE user LIKE "%山%";
```
# 4.完全一致検索
```SQL
SELECT user FROM users WHERE user LIKE "山";
```
# 5. 否定検索
```SQL
SELECT user FROM users WHERE user NOT LIKE "%山%";
```

## JSX:Javascriptにマークアップを入れ込む
JSXは言語の拡張であり、ReactはJavaScriptのライブラリ
中身のないタグ：==フラグメント==

```javascript
export default function Avatar() {
  const avatar = 'https://i.imgur.com/7vQD0fPs.jpg';
  const description = 'Gregorio Y. Zara';
  return (
    <img
      className="avatar"
      src={avatar}
      alt={description}
    />
  );
}
```
：波括弧を使うことでマークアップの中で直接javascriptを使えるようになる

## 波括弧はjavascript世界への窓口
：波括弧を使うことでマークアップの中で直接javascriptを使えるようになる

propsは==オブジェクト==、==配列==、==関数==などjavascriptのあらゆる値を渡すことができる

```javascript
function Avatar(props) {
  let person = props.person;
  let size = props.size;
  // ...
}
```
```javascript
function Avatar({ person, size }) {
  // ...
}
```
この二つのkーどは同じ意味を持つ。下が分割代入というやり方。
`<Avatar {...props} />` のような JSX スプレッド構文で==すべての props を転送できる==が、多用は禁物！
