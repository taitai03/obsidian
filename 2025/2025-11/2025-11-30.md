# SQL道場
## Null値の確認
```SQL
WHERE カラム名 IS Null
```

## 最大値と最小値の取得
```sql
SELECT MAX(カラム名), MIN(カラム名)
FROM <テーブル名>
```
asを使って新たにカラム名として指定することができる
```sql
SELECT MAX(カラム名) as 新しいカラム名,
	   MIN(カラム名) as 新しいカラム名
FROM テーブル名
```

## 範囲の指定
```sql
where カラム名 between A and B
```

## 特定の日付の取得
```sql
SELECT * FROM daily_retail_data WHERE transaction_date = '2024-01-01';
```
データが入ってる形に合わせる

```sql
SELECT AVG(daily_revenue) as avg_revenue FROM daily_retail_data;
```

## 業種カテゴリの件数
```sql
select count(distinct business_category) as category_count
from daily_retail_data
```

## GROUP BY でグループ化
```sql
SELECT business_category, SUM(daily_revenue) as total_revenue,  AVG(daily_revenue) as avg_revenue, 
MAX(daily_revenue) as max_revenue, 
MIN(daily_revenue) as min_revenue ,
FROM daily_retail_data 
GROUP BY business_category
```
