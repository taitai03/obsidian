
# 2.2Rubyに関する基礎知識

### 2.2.1全てがオブジェクト

Rubyは<span style="background:#fff88f">オブジェクト指向言語</span>。文字列や数値はもちろん、nilなどもオブジェクトである

### 2.2.2メソッド呼び出し

<span style="background:#fff88f">オブジェクト.メソッド(引数１,引数2,引数3)</span>
メソッドによって、引数がない場合がある。また括弧がなくても動作はする。

### 2.2.3文の区切り

Rubyは基本的に改行が文の区切り


### 2.3.1 シングルクォートとダブルクォート

基本的にこの二つに差はないが、\nなどの特殊文字を使うときにがはダブルクォートを使う必要がある。また、ダブルクォートを使うと、式展開することができる

```ruby
name="Alice"
puts = "HEllo #{name}."
```


# 2.4 数値

数値の中の_は無視される

```Ruby
100_000_000 #100000000
```


### 2.5.2 　論理演算子
<span style="background:#fff88f">&&</span>
```Ruby
#条件１も条件２も真であれば真、それ以外は偽
条件１ && 条件２
```
<span style="background:#fff88f">||</span>
```Ruby
#条件のいずれかが真なら真、どちらも偽なら偽
条件１ ||　条件２
```

<u><u>&&の優先度は||よりもたかい</u></u>

優先順位を変えたい時は、その条件に対して()を使う

### 2.5.3 if文
```ruby
if 条件A
	#条件Aが真だった場合の処理
elsif 条件B
	#条件Aが偽で、条件B真だった場合の処理
else
	#それ以外の処理
end
```
<span style="background:#fff88f">else節がなく、どの条件にも当てはまらなかった場合、nilが返る</span>。

# 2.6メソッドの定義
```ruby 
def メソッド名(引数１,引数２)
	#必要な処理
end
```
メソッド名は、小文字のスネークケースで書く。

# 2.8文字列についてもっと詳しく

### 2.8.1 文字列はStringクラスのオブジェクトである

### 2.8.3 ヒアドキュメント
```ruby
def some_method
	<<- TEXT
これはヒアドキュメントです
<<-を使うと最後の識別子をインデントさせることができる・
	TEXT
end

puts some_method
#=>	 これはヒアドキュメントです
#	   <<-を使うと内部文字のインデント部分が無視されます
```
TEXTの部分半でも良いが、慣例としてこれが使われている

### 2.8.4フォーマットを指定して文字列を作成する
sprintfメソッドは、指定されたフォーマットの文字列を作成するとができる
```ruby
sprintf('%0.3f',1.2) #=>1.200
```
### 2.8.5その他文字列の色々
```ruby
#数値を文字列に変換する
123.to_s #=>'123' 

#配列の各要素を結合して、一つの文字列にする
[10,20,30].join #=>'102030'
```


### 2.10.5条件演算子(三項演算子)
```ruby
式 ? 真だった場合の処理 : 偽だった場合の処理
```


# 2.11メソッドの定義にもっと詳しく
メソッドを呼び出すときに、引数の過不足があれば、エラーになる。そこで、デフォルト値を与えることで、このエラーを回避する。その時、<span style="background:#fff88f">デフォルト値は変数を取ることも可能である</span>
