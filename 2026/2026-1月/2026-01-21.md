## rails scopeの役割

検索条件を組み立てるための [[知らない単語#^01|糖衣構文]]

```ruby
scope :not_declined, -> {
  where.not(current_status: DECLINE_STATUSES)
}
```
これは、以下のように書いているのと同じ
```ruby
def self.not_declined
  where.not(current_status: DECLINE_STATUSES)
end
```
